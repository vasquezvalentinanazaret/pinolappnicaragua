# PinolApp - Plataforma de Delivery en Nicaragua

Aplicación de delivery de comida desarrollada con React Native Expo.

## Características

- 🍔 Catálogo de restaurantes
- 🛒 Carrito de compras
- 📍 Seguimiento de pedidos en tiempo real
- 💳 Procesamiento de pagos
- 👤 Perfil de usuario

## Requisitos previos

- Node.js >= 18
- npm o yarn
- Expo CLI
- Cuenta en Supabase
