# PinolApp Nicaragua

**App de delivery 100% nicaragüense**  
Comida típica como baho, gallo pinto, combo típico y más – directo a tu puerta en Managua y alrededores.

### Tecnologías planeadas
- Expo + Expo Router (navegación file-based)
- React Native
- Zustand (gestión de estado: carrito, pedidos)
- NativeWind / Tailwind CSS (estilos rápidos)
- Supabase (backend real-time para pedidos en vivo – lo conectamos después)

### Flujo principal de la app
1. Home: Restaurantes cercanos (El Buen Sabor, Súper Típico...)
2. Restaurante: Menú con items (Baho C$180, Combo Típico C$300...)
3. Carrito → Checkout (dirección, pago efectivo/tarjeta)
4. Tracking: Mapa con scooter, barra de progreso, info del repartidor

Colores principales: Verde pinolero #00A651, Rojo #E30613, Fondo claro.
